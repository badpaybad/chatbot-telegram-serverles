tôi muốn dùng IDE Google antigravity để vẽ và sửa các file auto cad.
hãy dựng 1 khung để tôi có thể bổ xung các:
- yêu cầu của tôi vào đó.
- các file tham khảo vào đó.
- các file yêu cầu vào đó.
- các mẫu toolcall dạng code có thể bổ xung thêm khi cần

khi có yêu cầu mới ở chat của IDE antigravity thì nó sẽ tìm đến khung trên tham khảo để vẽ hoặc chỉnh sửa file
khi làm tôi chỉ làm với 1 file active hiện tại. các lần chỉnh sửa sẽ được lưu ở folder processing. 
khi chốt được phương án cần tạo file sang folder done.


cần chạy trên ubuntu 24.04

**chú ý** cần bố trí folder bên trong folder gốc: agentic.auto.cad và tạo files, folders khi cần. Ghi và cập nhật các tìm hiểu, suy nghĩ, cách làm vào timhieu.md, sau này cập nhật thêm gì cũng phải ghi vào timhieu.md để tôi có thể xem lại và bổ xung thêm khi cần.
